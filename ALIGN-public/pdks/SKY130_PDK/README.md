# SkyWater 130 nm PDK adapter for ALIGN

This directory contains a SkyWater 130 nm PDK adapter for ALIGN. The base
files were derived from the ALIGN-pdk-sky130 repository
(<https://github.com/ALIGN-analoglayout/ALIGN-pdk-sky130>) and are used under
the same BSD-3-Clause license. The `cap.py` and `res.py` passive primitive
generators are custom versions maintained in this fork.
